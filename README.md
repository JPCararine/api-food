# algafood-api

Projeto em Java/Spring Boot inspirado em soluções como iFood e Uber Eats, objetivo do projeto é desenvolver uma API para uma empresa de delivery de comida.

## Requisitos

-JDK 17
-Maven
-MySQL[

## Principais Recursos

- [MySQL](https://www.mysql.com/)
- [Lombok](https://projectlombok.org/)
- [MapStruct](https://mapstruct.org/)
- [Flyway](https://flywaydb.org/)
- [JUnit](https://junit.org/)
